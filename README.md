# 요소수 유통·주유소 재고 현황 데이터 분석 프로젝트

AI 기반 요소수·휘발유 재고 관리 및 가격 정보 제공 플랫폼 **FUEL LINK** 제안을 위한 데이터 분석 미니 프로젝트입니다.

---

## 프로젝트 개요

2021년 요소수 대란 사례를 활용하여 주유소 경영 혁신 서비스를 분석하고 제안합니다.

**핵심 문제의식 (SCQA)**

- **Situation**: 전국 11,500개 주유소가 매일 재고와 가격 두 가지 핵심 자원을 관리해야 하나, 외부 환경에 극도로 취약
- **Complication**: 2021년 요소수 대란과 중동 유가 변동성 확대로 공급망 리스크 반복
- **Question**: 위기가 반복되는 시대에, 주유소 사장님은 어떻게 선제적으로 대응할 수 있을까?
- **Answer**: FUEL LINK — AI 기반 실시간 재고 관리 및 가격 비교 플랫폼

## 주요 분석 결과

- **전국 200개 주유소 분석**: 44%(88개소)가 재고 관리 위험 구간
- **수도권 집중**: 요소수 취급 주유소의 75%가 경기·인천 집중
- **가격-재고 독립성**: 피어슨 상관계수 r = 0.099 → 두 변수 별도 전략 필요
- **유형별 가격 차이**: 고속도로 휴게소(1,458원/L) vs 일반 주유소(1,539원/L), p < 0.05
